C++深入思考笔记
======================
### 学游戏开发，为什么要学C++？  
1. 牛人的游戏类库有很多都是C++写的，可以借鉴与引用；  
2. C++游戏软件可以体积很小；  
3. C++有指针，可以避免拷贝，软件执行效率高；  
4. C++手动回收垃圾，避免垃圾回收机制拖慢程序；  
5. UE4-5 使用C++；  
6. EOS区块链使用C++编写合约数据库；  
7. 比C语言多了面向对象和标准库；
8. 10年之内，CPU无法忽视垃圾回收机制；  

### 为什么很多人不想用C++   
1. 因为被懒惰而阴险的人给骗了，那些人说C++有多难、多难，其实自己懒得学，还怕别人学会了；   
2. 然后新手一开始选择了其他“初恋”，即使跟C++走到了一起，也老想着初恋；
