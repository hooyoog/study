C++深入思考笔记
======================
### 学游戏开发，为什么要学C++？  
1. 牛人的游戏类库有很多都是C++写的，可以借鉴与引用；  
2. C++游戏软件可以体积很小；  
3. C++有指针，可以避免拷贝，软件执行效率高；  
4. C++手动回收垃圾，避免垃圾回收机制拖慢程序；  
5. UE4-5 使用C++；  
6. EOS区块链使用C++编写合约数据库；  
7. 比C语言多了面向对象和标准库；
8. 10年之内，CPU无法忽视垃圾回收机制，C++是最快的；   
9. C++的哲学：“只为用到的东西付出代价”，  
   这是用C++的高手们的逼格，学习他们，参考他们有好处；

### 一. C++面向对象的一些思考  
1. 一个只有有参构造函数的类：  
如果 Piont p；//这样会出错的
没有无参构造函数，无法初始化对象状态  

	a. 无参语法错误  
	b. 无法给那些m_的成员数据赋予值，对象就没有数据状态  

2. 应该在何时使用delete ？     
	a. 函数必须有正确的析构函数，否则会调用错误的析构函数  
	b. 父类和父类struct必须有   
	c. 成员数据有指针   
		必须等一析构  
		并需要赋值构造函数  
		如果没有，将会浅拷贝，copy地址  
		多个对象使用一个数据  
		一个死掉 ， 其他对象都坏了  

3. copy构造函数，和赋值操作符  
	a. 记得参数加上const  
	b. 删除数组 delete[]

4. 有时候需要重写关系运算符，才能让+或者==打到你的目的
